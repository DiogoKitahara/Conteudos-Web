Questionário sobre CSS

*** Perguntas ***

O que significa CSS?

O signicado de CSS é Cascading Style Sheets ou Folha de Estilo em Cascata.

Cite 2 características importantes quando se usa CSS

Quando se usa CSS é importante se atentar aos 3 estilos e escolher qual usar. Além de estilizar de acordo com o W3C.

Quais as possíveis formas de vincular o CSS em um documento HTML? Cite e exemplifique.

As possíveis formas são incorporada que está incluído diretamento no cabeçalho do arquivo HTML, importada no qual os estilos estão em um arquivo próprio de CSS e inline quando está dentro das tags do arquivo HTML.

Na definição do CSS o que representa a criação de uma regra no seletor?

Em CSS o que representa a criação de uma regra no seletor é "seletor {propriedade-css: valor}"

Como se comenta uma linha na definição do arquivo CSS?

Para comentar uma linha na definição do arquivo do CSS é "/* ... */"

Quais são os delimitadores de início e fim na definição de uma regra do CSS?

Os delimitadores são "{ }"

Quando o ; será utilizado?

O ";" será utilizado quando terminar de declarar uma propriedade

O que são valores múltiplos para serem usados nas regras CSS?

Os valores múltiplos são usados para especificar mais de um valor para um propriedade
